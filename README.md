# yellowstone-grpc-golang
可快速上手的 Yellowstone gRPC  golang 教程

## 使用说明
使用之前你需要安装如下库
```
	google.golang.org/grpc v1.67.1
	google.golang.org/protobuf v1.35.1
    github.com/rpcpool/yellowstone-grpc/examples/golang/proto
```
